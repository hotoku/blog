開いているpaneを、他のセッション移す方法。

`move-pane`というコマンドがある。

```
<prefix>-: move-pane -t セッション番号:ウィンドウ番号
```

で移すことが可能。セッション番号とウィンドウ番号を`:`で区切って指定するのは、tmuxでの標準的な指定方法っぽい。

セッション番号は、tmuxのステータスバーの一番左に、`[ ]`に挟まれて表示されている。

```
[1] 0:home* 1:tmp 2:etc
```

みたいな感じ。
